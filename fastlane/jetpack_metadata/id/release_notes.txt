- Pembaruan blok sematan: Kami telah mengaktifkan pratinjau sematan WordPress, serta sematan untuk konten Instagram dan Vimeo. Untuk beberapa media, Anda mungkin berhasil menyematkan URL tetapi tidak dapat melihat pratinjau konten; dalam hal ini, Anda akan melihat pesan “Pratinjau tidak tersedia”. Jika Anda tidak dapat menyematkan media, Anda akan melihat pilihan "Coba lagi" untuk menyematkan konten kembali, dan pilihan alternatif untuk mengonversinya menjadi tautan.
